Lambda表达式

lambda表达式是一个可传递的代码块，可以在以后执行一次或多次。

用lambda表达式创建多线程

创建线程有多种方法，继承Thread类，实现Runnable接口，或者能直接通过一个匿名内部类。而我们只是向把run方法中的代码块传递给线程，告知线程做什么事。

Lambda标准格式

  • Lambda：(参数类型 参数名称)->{代码块} 如果参数没有参数就为空

省略格式

Lambda表达式可以根据上下文自己推导。

  • 可以省略的内容：
    • (参数列表)中的数据类型可以省略不写
    • (参数列表)中如果只有一个，那么类型和括号都可以省略
    • {代码块}如果括号中的代码只有一行且无论是否有返回值，都可以省略括号，return，分号（要省一起省）
