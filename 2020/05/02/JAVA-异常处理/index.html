ејӮеёёеӨ„зҗҶ

еҒҮи®ҫдёҖдёӘзЁӢеәҸиҝҗиЎҢжңҹй—ҙеҮәзҺ°дәҶй”ҷиҜҜгҖӮзЁӢеәҸеә”иҜҘпјҡ

  • иҝ”еӣһдёҖз§Қе®үе…ЁзҠ¶жҖҒпјҢе№¶иғҪеӨҹи®©з”ЁжҲ·жү§иЎҢдёҖдәӣе…¶д»–зҡ„е‘Ҫд»Ө
  • е…Ғи®ёз”ЁжҲ·дҝқеӯҳжүҖжңүж“ҚдҪңз»“жһңпјҢе№¶дё”д»ҘеҰҘе–„зҡ„ж–№ејҸз»ҲжӯўзЁӢеәҸ

ејӮеёёеҲҶзұ»

еңЁJavaдёӯпјҢејӮеёёеҜ№иұЎйғҪжҳҜжҙҫз”ҹдәҺThrowableзұ»зҡ„дёҖдёӘе®һдҫӢгҖӮжүҖжңүзҡ„ејӮеёёйғҪжҳҜз”ұThrowableз»§жүҝиҖҢжқҘпјҢдҪҶеңЁдёӢдёҖеұӮз«ӢеҚіеҲҶи§ЈдёәдёӨдёӘеҲҶж”Ҝпјҡerrorе’ҢExceptionгҖӮ
Errorзұ»еұӮж¬ЎжҸҸиҝ°дәҶJavaиҝҗиЎҢж—¶зі»з»ҹзҡ„еҶ…йғЁй”ҷиҜҜе’Ңиө„жәҗиҖ—е°Ҫй”ҷиҜҜгҖӮеә”з”ЁзЁӢеәҸдёҚеә”иҜҘжҠӣеҮәиҝҷз§Қзұ»еһӢзҡ„еҜ№иұЎгҖӮ
ExceptionеұӮж¬ЎеҸҲеҲҶдёәдёӨдёӘеҲҶж”ҜпјҡIOExceptionе’ҢRuntimeExceptionгҖӮз”ұзЁӢеәҸй”ҷиҜҜеҜјиҮҙзҡ„ејӮеёёеұһдәҺRuntimeExceptionпјӣиҖҢзЁӢеәҸжң¬иә«жІЎжңүй—®йўҳпјҢз”ұдәҺI/Oй”ҷиҜҜиҝҷзұ»й—®йўҳеҜјиҮҙзҡ„ејӮеёёеұһдәҺе…¶д»–ејӮеёёгҖӮ
RuntimeExceptionй”ҷиҜҜжҲ‘д»¬еҸҜд»ҘдёҚеҜ№е…¶иҝӣиЎҢеӨ„зҗҶпјҢеҸӘжҳҜжҠӣеҮәпјҢдәӨз»ҷJVMеӨ„зҗҶгҖӮ

ејӮеёёеӨ„зҗҶ

throwableдёӯе®ҡд№үдәҶдёүдёӘејӮеёёеӨ„зҗҶзҡ„ж–№жі•

ж–№жі•еҗҚ дҪңз”Ё
String getMessage() иҝ”еӣһжӯӨејӮеёёзҡ„з®ҖзҹӯжҸҸиҝ°
String toString() иҝ”еӣһжӯӨејӮеёёзҡ„иҜҰз»ҶдҝЎжҒҜеӯ—з¬ҰдёІпјҲдёҖиҲ¬дёҚз”Ёпјү
void printStackTrace() е°ҶејӮеёёиҝҪиёӘиҫ“еҮәиҮіж ҮеҮҶй”ҷиҜҜжөҒ

throw

еңЁJavaдёӯпјҢжҸҗдҫӣдәҶдёҖдёӘе…ій”®еӯ—$\color{blue}{throw}гҖӮе®ғз”ЁжқҘжҠӣеҮәдёҖдёӘжҢҮе®ҡзҡ„ејӮеёёеҜ№иұЎгҖӮжӯҘйӘӨпјҡ

  • еҲӣе»әдёҖдёӘејӮеёёеҜ№иұЎпјҢе°ҒиЈ…дёҖдәӣжҸҗзӨәдҝЎжҒҜ
  • йңҖиҰҒе°ҶиҝҷдёӘејӮеёёеҜ№иұЎе‘ҠзҹҘз»ҷи°ғз”ЁиҖ…
    throwз”ЁеңЁж–№жі•еҶ…пјҢз”ЁжқҘжҠӣеҮәдёҖдёӘејӮеёёеҜ№иұЎгҖӮе°ҶиҝҷдёӘејӮеёёеҜ№иұЎдј йҖ’еҲ°и°ғз”ЁиҖ…еҮәпјҢе№¶з»“жқҹеҪ“еүҚж–№жі•зҡ„жү§иЎҢгҖӮ
    иҜӯжі•пјҡ
    throw new ејӮеёёзұ»еҗҚ(еҸӮж•°);

еЈ°жҳҺејӮеёёthrows

  • еЈ°жҳҺејӮеёёпјҡе°Ҷй—®йўҳж ҮиҜҶеҮәжқҘпјҢжҠҘе‘Ҡз»ҷи°ғз”ЁиҖ…гҖӮеҰӮжһңж–№жі•еҶ…йҖҡиҝҮдәҶthrowжҠӣеҮәдәҶе…¶д»–ејӮеёёпјҢиҖҢжІЎжңүиҝӣиЎҢжҚ•иҺ·еӨ„зҗҶпјҢйӮЈд№Ҳеҝ…йЎ»йҖҡиҝҮthrowsиҝӣиЎҢеЈ°жҳҺпјҢдәӨз»ҷи°ғз”ЁиҖ…еӨ„зҗҶгҖӮ
    еЈ°жҳҺејӮеёёж јејҸпјҡ
    дҝ®йҘ°з¬Ұ иҝ”еӣһеҖјзұ»еһӢ ж–№жі•еҗҚ(еҸӮж•°еҲ—иЎЁ) throws ејӮеёёзұ»еҗҚ1пјҢ ејӮеёёзұ»еҗҚ2 вҖҰвҖҰ

жҚ•иҺ·ејӮеёёtry-catch

еҰӮжһңжҹҗдёӘејӮеёёеҸ‘з”ҹзҡ„ж—¶еҖҷе№¶жІЎжңүеңЁд»»дҪ•ең°ж–№иҝӣиЎҢжҚ•иҺ·пјҢйӮЈзЁӢеәҸе°ұдјҡз»Ҳжӯўжү§иЎҢпјҢе№¶еңЁжҺ§еҲ¶еҸ°дёҠжү“еҚ°еҮәејӮеёёдҝЎжҒҜпјҢе…¶дёӯеҢ…жӢ¬ејӮеёёзҡ„зұ»еһӢе’Ңе Ҷж Ҳзҡ„еҶ…е®№гҖӮ
try-catchиҜӯжі•пјҡ


еҰӮжһңеңЁtryиҜӯеҸҘеқ—дёӯзҡ„д»»дҪ•д»Јз ҒжҠӣеҮәдәҶдёҖдёӘеңЁcatchеӯҗеҸҘдёӯиҜҙжҳҺзҡ„ејӮеёёзұ»пјҢйӮЈд№Ҳпјҡ

  • зЁӢеәҸе°Ҷи·іиҝҮtryиҜӯеҸҘеқ—зҡ„е…¶дҪҷд»Јз ҒгҖӮ
  • зЁӢеәҸе°Ҷжү§иЎҢcatchеӯҗеҸҘдёӯзҡ„ејӮеёёеӨ„зҗҶд»Јз ҒгҖӮ
  • finallyдёӯзҡ„д»Јз Ғж— и®әжңүжІЎжңүејӮеёёйғҪдјҡиў«жү§иЎҢгҖӮ
  • еҰӮжһңfinallyдёӯжңүreturnиҜӯеҸҘпјҢиҝ”еӣһзҡ„ж°ёиҝңж—¶finallyдёӯзҡ„еҖј

еҰӮжһңеңЁtryиҜӯеҸҘеқ—дёӯжІЎжңүжҠӣеҮәд»»дҪ•ејӮеёёпјҢйӮЈд№ҲзЁӢеәҸе°Ҷи·іиҝҮcatchеӯҗеҸҘгҖӮcatchеҸҜд»ҘжңүеӨҡеҸҘпјҢжҚ•иҺ·еӨҡдёӘејӮеёё

еӨҡдёӘејӮеёёжҚ•иҺ·ж—¶зҡ„жіЁж„ҸдәӢйЎ№пјҡ

  • catchйҮҢйқўе®ҡд№үзҡ„ејӮеёёеҸҳйҮҸпјҢеҰӮжһңжңүеӯҗзҲ¶зұ»зҡ„е…ізі»пјҢеӯҗзұ»зҡ„ејӮеёёеҸҳйҮҸеҝ…йЎ»еҶҷеңЁзҲ¶зұ»ејӮеёёеҸҳйҮҸд№ӢеүҚ
  • зҲ¶зұ»ж–№жі•жІЎжңүжҠӣеҮәејӮеёёпјҢеӯҗзұ»иҰҶзӣ–зҲ¶зұ»ж–№жі•ж—¶д№ҹдёҚеҸҜжҠӣеҮәејӮеёёгҖӮжӯӨж—¶еӯҗзұ»еҮәзҺ°ејӮеёёпјҢеҸӘиғҪдҪҝз”Ёtry-catchиҝӣиЎҢеӨ„зҗҶ

иҮӘе®ҡд№үејӮеёё

JDKжүҖжҸҗдҫӣзҡ„ејӮеёёзұ»еһӢжңүйҷҗпјҢеңЁејҖеҸ‘зҡ„йңҖжұӮдёӯеҸҜд»Ҙж №жҚ®иҮӘе·ұзҡ„йңҖиҰҒеҺ»е®ҡд№үиҮӘе·ұзҡ„ејӮеёёзұ»гҖӮ

  1. иҮӘе®ҡд№үдёҖдёӘзј–иҜ‘жңҹејӮеёёпјҡиҮӘе®ҡд№үзұ»з»§жүҝдәҺExceptionгҖӮ
  2. иҮӘе®ҡд№үдёҖдёӘиҝҗиЎҢж—¶жңҹејӮеёёпјҡиҮӘе®ҡд№үзұ»з»§жүҝдәҺRuntimeExceptionгҖӮ
