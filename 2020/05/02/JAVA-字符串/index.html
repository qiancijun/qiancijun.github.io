String

  • String是一个final类，代表不可变的字符序列
  • 字符串是常量，用双引号来表示，他们的值在创建之后不可被改变。
  • 源码：
  • 特点：
    1. 底层是一个字符数组
    2. String声明为final，不可被继承
    3. String实现了Serializable接口，表示字符串是可以被序列化的
    4. 实现了Comparable接口，表示String可以比较大小
    5. 通过字面量的方式（区别于new方式）给一个字符串赋值，此时的字符串声明在字符串常量池中。当对字符串重新赋值时，需重写指定内存区域赋值，不能在原有的value赋值
    6. 字符串常量池中是不会存储相同的字符串的

¶对象创建

  • 源码
  • String str1 = "abc"; 与 String str2 = new String("abc"); 的区别
    • 字符串常量存储在字符串常量池，目的是共享。字符串非常量对象存储在堆中。
    • str1在栈空间种，str2在堆空间种
  • String s = new String("hello); 方式创建对象，在内存中创建了几个对象
    • 两个：一个在堆空间中new String创建的对象，第二个是char型数组在对应的常量池中
    • 如果常量池有了，只会创建一个

¶字符串拼接

  • 常量与常量的拼接结果在常量池。且常量池中不会存在相同内容的常量。
  • 只要其中有一个是变量，结果就在堆中
  • 如果拼接的结果调用intern()方法，返回值就在常量池中。

¶类型转换

  • String --> char[]
  • char[] --> String
  • String --> byte[]
  • byte[] --> String
