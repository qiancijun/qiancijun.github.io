集合

  • java.util.List接口继承自Collection接口，是单列集合的一个重要分支。
    本文只记录了具体集合的用法，这些集合怎么实现的，是数据结构课程的内容，本文不给予解释（篇幅太长）。

    List

方法名 作用
void add(E newElement) 在当前位置前添加一个元素
void addAll(int i, Collection<? extends E> elements) 将某个集合中的所有元素添加到指定位置
E remove(int i) 删除给定位置的元素并返回这个元素
E get(int i) 获取给定位置的元素
E set(int i, E element) 用新元素取代给定位置的元素，并返回原来那个元素
int indexOf(Object element) 返回与之前元素相等的元素在列表中第一次出现的位置，如果没有返回-1
void set(E newElement) 用新元素取代next或previous上次访问的元素。如果在next或previous上次调用之后列表结构被修改了，将抛出一个IllegalStateException异常
boolean hasPrevious() 当反向迭代列表时，还有可供访问的元素，返回true
E previous() 返回前一个对象。如果已经到达了列表的头部，就抛出一个NoSuchElementException异常
int nextIndex() 返回下一次调用next方法时将返回的元素索引
int previousIndex() 返回下一次调用previous将返回的元素索引

ArrayList

ArrayList的结构是数组结构。元素增删慢，查找快。$\color{red}{此实现不是同步的}$。如果多个线程同时访问一个ArrayList实例，而其中至少一个线程从结构上修改了列表，那么它对外必须保持同步。

LinkedList

LinkedList的结构时链表结构。元素增删快。双向链表

方法名 作用
void addFirst(E element) 添加元素
void addLast(E element) 添加元素
同样的有remove方法和get方法，pop等效于removeFirst，push等效于addFirst

Queue

特点：先进先出
Queue<Integer> q = new LinkedList<>();

方法名 作用
void add(E element) 在队尾添加元素
E element() 获取队头元素，不会移除队头元素，若队列为空会抛出异常
E peek() 获取队头元素，不会移除队头元素，队列为空不会抛出异常
E remove() 移除队头元素并且返回该元素，队列为空会抛出异常
E poll() 移除队头元素并且返回该元素，队列为空不会抛出异常

PriorityQueue

优先队列:PriorityQueue<Integer> pq = new PriorityQueue<>();

PriorityQueue(Comparator<? super E> comparator)
Creates a PriorityQueue with the default initial capacity and whose elements are ordered according to the specified comparator.

使用样例，可以new一个Compartor，也可以直接使用Lambda表达式。

Stack

特点：先进后出
Stack<Integer> s = new Stack<>();

方法名 作用
E push(E item) 压栈
E pop() 出栈并且返回该元素
E peek() 返回堆顶元素
boolean empty() 栈是否为空

set

没有重复元素，没有下标索引（无序）。查询速度快。

HashSet

HashSet<Integer> s = new HashSet<>();
