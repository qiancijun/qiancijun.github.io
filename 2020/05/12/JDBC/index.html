什么是JDBC

  • JDBC：Java Database Connectivity java和数据库的连接技术，sun公司专门推出的一套java应用程序访问数据库的技术规范。
  • 规范：抽象类和接口（接口居多）
    在Java中，数据库存取技术可分为几类：
  1. JDBC直接访问数据库
  2. JDO技术（Java Data Object）
  3. 第三方O/R工具，如Hibernate,Mybatis等

JDBC的使用

注册驱动

首先来回顾一下类的加载方式：

  1. new 对象
  2. 加载子类会导致父类的加载
  3. 调用类中的静态成员
  4. 通过反射

使用new对象加载方式的不足：

  1. 属于编译期加载，如果编译期间该类不存在，则直接报编译错误，依赖性太强
  2. 导致Driver对象创建了两遍，效率较低

采用反射的方式加载类：
Class.forName(全类名)

  1. 属于运行期加载，大大降低了类的依赖性

  2. Driver对象仅仅创建了一遍，效率较高

  3. class.forNmae

  4. 创建driver对象

获取连接

DriverManager.getConnection有三种重载，返回一个connection对象。

  1. DriverManager.getConnection(“jdbc:mysql//localhost:3306/库名?user=root&password=密码”);

  2. DriverManager.getConnection(“jdbc:mysql//localhost:3306/库名”,Properties info);
