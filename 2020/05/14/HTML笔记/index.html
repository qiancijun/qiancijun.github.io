元数据

  • meta：主要用于设置网页中的一些元数据，元数据不是给用户看
    • charset: 指定网页的字符集
    • name: 指定的数据的名称
    • content: 指定数据的内容
      • keywords: 表示网站的关键字，可以同时指定多个关键字，关键字使用逗号隔开
      • description: 用于指定网站的描述

语义化标签

在网页中HTML专门用来负责网页的结构，所以在使用HTML标签时，应该关注的是标签的语义，而不是它的样式。（样式完全可以由CSS修改）。

  • 块元素：在页面中独占一行的元素称为块元素(block element)。
  • 行元素：在页面中不会独占一行的元素称为行内元素(inline element)。

    标题标签

    <h1></h1>:一级标题，是块元素
    h1 ~ h6一共有六级标题，重要性递减，h1最重要，h6最不重要。h1在网页中的重要性仅次于title标签。一般情况下一个页面中只会有一个h1，标题标签只用到h3，h4 ~ h6很少用。

p标签

  • <p></p>：p标签表示页面中的一个段落，是块元素。

hgroup标签(HTML5)

  • <hgroup></hgtoup>:用来为标题分组，可以将一组相关的标题同时放入到hgroup。

em标签

  • <em></em>: 语气加重，是行元素。

strong标签

  • <strong></strong>：表示强调。

blockquote

  • <blockquote></blockquote>: 表示一个长引用，是块元素

br

  • <br>: 页面中的换行

块和行元素

  • 块元素：
    • 在网页中一般通过块元素来对页面进行布局
  • 行内元素：
    • 主要用来包裹文字

注：

  1. 一般情况下会在块元素中方行内元素，而不会在行内元素中放块元素。
  2. 块元素中基本上什么都能放。
  3. p元素中不能放任何的块元素。
  4. 浏览器在解析网页时，会自动对网页中不符合规范的内容进行修正，比如：
    • 标签写在了根元素的外部
    • p元素中嵌套了块元素
    • 根元素中出现了除head和body以外的子元素

结构化语义标签(HTML5)

在网页中使用并不是很多，用的最多的还是div和span。

header

  • <header></header>：表示网页的头部，也可以为顶部。

main

  • <main></main>：表示网页的主题（一个页面中只会有一个main）。

footer

  • <footer></footer>：表示网页的底部。

nav

  • <nav></nav>：网页的导航

aside

  • <aside></aside>：和主体相关的其他内容，但不属于主体（侧边栏）

article

  • <article></article>：表示一个独立的文章

section

  • <section></section>：表示一个独立的区块，上面的标签都不能表示时，使用section

div

  • <div></div>：没有语义，表示一个区块，主要的布局元素

span

  • <span></span>：行内元素，没有任何的语义，一般用于在网页中选中文字

列表

  • 列表(list)：在html中分为了三种：有序列表和无序列表在不同浏览器显示效果会不同，所以很少使用，列表之间可以互相嵌套
    • 有序列表：
      • 使用ol标签来创建有序列表<ol></ol>
      • 使用li来表示列表项<li></li>：如<li>选项</li>
    • 无序列表：
      • 使用ul标签来创建无序列表<ul></ul>
      • 使用li来表示列表项<li></li>：如<li>选项</li>
    • 定义列表：
      • 使用dl标签来创建定义列表
      • 使用dt来表示定义的内容
      • 使用dd来对内容进行解释说明

超链接

  • 超链接可以让我们从一个页面跳转到其他页面，或者是当前页面的其他位置。使用a标签来定义超链接。超链接是一个行内元素，在a标签中可以嵌套任何的元素，除了它自身。

属性

href

  • 指定跳转的目标路径，值可以是一个外部网站的地址，也可以写一个内部页面的地址。如果把href属性设置为#，这样点击超链接以后，页面不会发生跳转，而是转到当前页面的顶部。如果设置为javascript:;，此时点击超链接什么也不会发生。

target

  • 指定超链接打开的位置，可选值：
    • _self:默认值，在当前页面中打开超链接
    • _blank:在一个新的页面中打开超链接

图片标签

  • <img>:用于向当前页面中引入一个外部图片，img是一个自结束标签。是一个替换元素（基于块和行内元素之间，具有两种元素的特点）
    • src：图片的路径
    • alt：图片的描述，默认情况下不会显示，有些浏览器会在图片无法加载时显示。搜索引擎会根据alt中的内容来识别图片人，如果不写alt属性则图片不会被搜索引擎所识别。
    • width：图片的宽度（像素）
    • height：图片的高度（像素）如果高度和宽度只修改了一个，则另一个会等比例缩放。

注：一般在pc端，不建议修改图片的大小。直接改变图片大小。
但是在移动端，经常需要对图片进行缩放

内联框架

  • <iframe></iframe>：用于向向当前页面中引入一个其他页面，现在内联框架使用的机会不多
    • src：指定要有引入的网页路径
    • width：宽
    • height：高
    • frameborder：是否有边框

音视频播放

音视频文件引入时，默认情况下不允许用户自己控制播放停止。

  • <audio></audio>: 向页面中引入一个外部的音频文件
    • src: 文件路径
    • controls: 是否允许用户控制播放，不用指定值。
    • autoplay：是否允许自动播放。如果设置了autoplay则音乐在打开页面时会自动播放，但是目前来讲大部分浏览器都不会对音乐自动播放。
    • loop：是否允许循环播放

除了通过src来指定外部文件的路径以外，还可以通过source来指定文件，IE8不支持音视频播放，用下种写法可以告诉用户不支持音视频播放。

注：source可以同时指定多个路径，如果第一个路径失效会选择第二个路径

  • <vedio></vedio>: 向页面中引入一个视频文件，和音频插入基本相同
