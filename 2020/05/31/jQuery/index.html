区别

  • 原生JS会等到DOM元素加载完毕，才会执行。如果有多个入口函数，后编写会覆盖之前的。
  • jQuery会等到DOM元素加载完毕，但是不会等到图片也加载完就会执行。如果有多个入口函数，不会覆盖。

其他写法

冲突问题

  • 如果其它框架也有$符号，会造成$符号的多义
  • 解决方案：
    • 方案一：释放$的使用权 jQuery.noConflict();
    • 方案二：自定义一个访问符号let nj = jQuery.noConflict;

静态方法和实例方法

静态方法

each

  • 原生js：原生的forEach方法只能遍历数组，不能遍历伪数组
  • jQuery：与原生js传入的参数不同，可以遍历伪数组

map

  • 原生js：不能遍历伪数组

  • jQuery：可以遍历伪数组

  • jQuery中map与each的区别：

    • each静态方法默认的返回值，遍历谁返回谁，不支持在回调函数中对遍历的数组进行处理
    • map静态方法的返回值是一个空数组，可以在回调函数中通过return对遍历的数组进行处理，生成一个新的数组

trim

  • 去除字符串中两端的空格

isArray

  • 判断传入的对象是不是数组对象，返回一个布尔值，伪数组为false，只有真数组为true

isWindow

  • 判断传入的对象是不是window对象，返回一个布尔值

isFunction

  • 判断传入的对象是不是一个函数，

    注：jQuery本质是一个函数

holdReady

  • 暂停ready的执行，启用ready执行$.holdReady(false);

选择器

内容选择器

  1. :empty：找到没有文本内容没有子元素的指定元素

  2. :parent：找到有文本内容或有子元素的指定元素

  3. :contains：找到包含指定文本内容的指定元素

  4. has：找到包含指定子元素的指定元素

操作属性节点

attr

  • 获取，设置属性节点的值。可以传递一个或两个参数。如果传递一个参数代表获取属性节点的值，两个则代表设置属性节点的值

    注：无论找到多少个元素，都会返回第一个元素指定的属性节点的值

注：如果是设置，则找到多少个，就设置多少个。如果没有该属性节点，就新增一个属性节点

removeAttr

  • 删除属性节点

    注：会删除所有找到元素的属性节点

操作属性

prop

  • 特点和attr一致
  • 不仅能够操作属性，还能够操作属性节点。在具有true和false的属性节点用prop，其余用attr

    removeProp

  • 特点和removeAttr一致

jQuery操作class

addClass

  • 添加一个类，如果要添加多个，多个类名之间用空格隔开
    $("div").addClass("class1 class2");

    removeClass

  • 删除一个类，如果要删除多个，多个类名之间用空格隔开
    $("div").removeClass("class2");

    toggleClass

  • 切换类，如果有就删除，没有就添加
    $("div").toggleClass("class1");

文本值操作

布局:


html

  • 和原生js中innerHTML一模一样
    $("div").html("<p>我是p标签</p>");
  • 获取HTML
    let $html = $("div").html();

    text

  • 设置文本内容
    $("div").text("hello world");
  • 获取文本内容
    let $text = $("div").text();

    val

  • 设置input的value
    $("input").val("hello world");
  • 获取input的value
    let $input = $("input").val();

事件

事件绑定

  1. 通过eventName(fn)：编码效率较高，有的js事件添加不上。多个绑定不会被覆盖
  2. 通过on(eventName,fn)推荐用on，后面的会覆盖前面的

事件解绑

  • off()：如果不传递参数，移除所有事件
  • off("click", method)：移除指定类型的指定事件

事件自动触发

  • trigger():事件自动触发，会触发事件冒泡，会触发默认行为
  • triggerHandler自动触发事件，不会触发事件冒泡，不会触发默认行为

    注：如果想对a标签用trigger建议在a中嵌套span

自定义事件

  1. 自定义事件只能通过on来绑定
  2. 事件必须通过trigger()来触发

    事件命名空间

  • 事件必须通过on来绑定，在on绑定的事件后面加上.name，再通过自动触发来调用

移入移出事件

  • mouseover()：鼠标移入事件
  • mouseout()：鼠标移出事件
  • 移入移出事件都会触发事件的冒泡，jQuery提供了另外两个方法
  • mouseenter()：鼠标移入事件
  • mouseleave()：鼠标移出事件
  • hover(func1, func2)：鼠标移入移出事件，第一个参数为移入函数，第二个参数为移出函数。如果只传入一个则即监听移入也监听移出。

  1. slideToggle

  • 淡入动画，第一个参数为毫秒数，第二个参数为回调函数

节点

  1. 内部插入
    • append：将指定元素添加到元素的最后
    • prepend：将指定元素添加到元素的最前
    • appendTo：将指定元素添加到元素的最后（调用者不同）
    • prependTo：将指定元素添加到元素的最前（调用者不同）
  2. 外部插入
    • after：将指定元素添加到元素外部的后面
    • insertAfter：将指定元素添加到元素外部的后面（调用者不同）
    • before：将指定元素添加到元素外部的前面
    • insertBefore：将指定元素添加到元素外部的前面（调用者不同）
  3. 删除节点
    • remove()：删除指定元素
    • empty()：删除指定元素的内容和子元素，指定元素自身不会被删除
  4. 替换节点
    • replaceWith：替换所有匹配的元素为指定的元素
    • replaceAll：替换所有匹配的元素为指定的元素（调用者不同）
  5. 复制节点
    • clone
      • 浅复制节点：传递false就为浅复制
      • 深复制节点：传递true就为深复制，会连同事件一同复制
