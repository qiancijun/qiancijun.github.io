CSS

网页分为三部分：

  1. 结构（HTML）
  2. 表现（CSS）
  3. 行为（JavaScript）
  • CSS：
    • 层叠样式表，修改每一个标签长什么样子
    • 网页实际上是一个多层的结构，通过CSS可以分别为网页的每一个层来设置样式而最终我们能看到的只是网页的最上边一层
    • CSS用来设置网页中元素的样式

例：使用CSS来修改元素的样式

  1. 在标签内部通过style属性来设置元素的样式（内联样式，行内样式）在实际开发中不建议使用

    style="color: darkmagenta; font-size: 200px;"：这句话就已经属于CSS样式了。使用内联样式只能对一个标签生效，如果希望影响到多个元素，必须在每个元素中都加上style。

  2. 将样式编写到head中的style标签里（内部样式表）：
    将样式编写到head中的style标签里，然后通过CSS的选择器来选中元素并将其设置各自样式。
    可以同时为多个标签设置样式，并且修改时只需要修改一处即可。

    但是内部样式表只能对一个网页起作用。不能跨页面进行复用

  3. 可以将CSS样式编写到外部css文件中（外部样式表）
    通过link标签引入外部的css文件。外部样式表需要通过link标签进行引入，意味着只要想使用这些样式的网页都可以对其进行引用。可以使用浏览器的缓存机制加快网页的加载速度，提高效率。

关系选择器

  • 父元素
    • 直接包含子元素的元素
  • 子元素
    • 直接被父元素包含的元素
  • 祖先元素
    • 直接或间接包含元素的元素
    • 一个元素的父元素也是它的祖先元素
  • 后代元素
    • 直接或间接被祖先元素包含的元素
    • 子元素也是后代元素
  • 兄弟元素
    • 拥有相同父元素的元素是兄弟元素

伪类选择器

  • 伪类：不存在的类，是一个特殊的类。用来描述一个元素的特殊状态。伪类都是依据所有的子元素进行排序的。
    • 一般情况下都是使用:开头
    • :first-child：第一个子元素
    • last-child：最后一个子元素
    • nth-child()：任意第n个子元素，如果参数填n则全选子元素。2n选中偶数位的子元素。或者用even，odd表示奇偶
    • first-of-type：不同类型的第一个子元素
    • last-of-type：不同类型的最后一个子元素
    • nth-of-type：不同类型第n个子元素
    • :not()：否定伪类，将符合条件的元素从选择器中去除

超链接的伪类

  • link：用来表示没有访问过的连接（正常的连接）
  • visited：用来表示访问过的连接（运用的不多）
  • hover:用来表示鼠标移入的状态（不仅仅只有超链接能用）
  • active：用来表示鼠标点击后的状态
