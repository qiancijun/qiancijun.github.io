¶浮点数二分

前缀和差分

  • 一维前缀和 S[i] = a[1] + a[2] + ... a[i]
  • 二维前缀和 S[i, j] = 第i行j列格子左上部分所有元素的和以(x1, y1)为左上角，(x2, y2)为右下角的子矩阵的和为：S[x2, y2] - S[x1 - 1, y2] - S[x2, y1 - 1] + S[x1 - 1, y1 - 1]
  • 一维差分：给区间[l, r]中的每个数加上c：B[l] += c, B[r + 1] -= c
  • 二维差分：给以(x1, y1)为左上角，(x2, y2)为右下角的子矩阵中的所有元素加上c：S[x1, y1] += c, S[x2 + 1, y1] -= c, S[x1, y2 + 1] -= c, S[x2 + 1, y2 + 1] += c
  • 例题：
    1. k倍区间

¶最短路问题

  • 单源最短路：

    1. 所有权值边都是正数：
      • 朴素Dijkstra算法 $O(n^{2})$
      • 堆优化版Dijkstra算法 $O(mlogn)$
    2. 存在负权值边
      • Bellmen-Ford $O(nm)$
      • SPFA 一般$O(m)$，最坏$O(nm)$

  • 多源最短路

    1. Floyd算法 $O(n^{3})$

  • 例题

    1. Dijkstra求最短路 I
    2. Dijkstra求最短路 II
    3. 有边数限制的最短路
    4. spfa求最短路
    5. spfa判断负环
    6. Floyd求最短路
