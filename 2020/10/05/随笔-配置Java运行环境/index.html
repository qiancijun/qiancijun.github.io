准备jdk

  • oracle官网，需要注册账号，下载jdk
  • 推荐下载jdk8，jdk11，jdk14

安装jdk

  • 为jdk分配一个文件夹，这里建议专门指定一个位置，安放所有的环境，方便自己寻找

配置环境变量

  • 配置用户环境和系统环境都可以，个人电脑无所谓

测试

  • Win+r输入cmd快速打开命令行窗口，输入java -version检查是否配置成功，对比一下输出结果，是否有java版本，java运行环境和JVM等信息

测试Java程序

编译器选择

  • 列举一些，仅供参考，排名不分先后
  1. Visual Studio Code（更像是文编编辑器，需要在社区安装Java运行插件，仅作测试，随手写一个Java程序时推荐使用，软件比较轻量级）
  2. Eclipse（老牌编译器）
  3. IDEA（JB公司家的编译器都不错，做项目首选，平时使用推荐使用社区版）
  4. spring tools suite(STS) （Spring家族包装的编译器，其实是对Eclipse的二次包装）
