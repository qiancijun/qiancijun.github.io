反射

反射(Reflection)是被视为动态语言的关键，反射机制允许程序在执行期间借助于Reflection API取得任何类的内部信息，并且能够直接操作任意对象的内部属性及方法。这种动态获取信息以及动态调用对象方法的功能称为java语言的反射机制。

反射机制功能

  1. 在运行时判断任意一个对象所属的类
  2. 在运行时构造任意一个类的对象
  3. 在运行时判断任意一个类所具有的成员变量和方法
  4. 在运行时调用任意一个对象的成员变量和方法
  5. $\color{red}{生成动态代理}$

Class类

在Object类中定义了以下的方法，此方法将被所有子类继承。

public final Class getClass()
方法的返回值是一个Class类，此类是Java反射的源头。

反射可以得到的信息：某个类的属性、方法、实现了哪些接口。对于每个类而言，JRE都为其保留一个不变的Class类型的对象。Class本身也是一个类，Class对象只能由系统建立。一个类在JVM中只会有一个Class实例，一个Class对象对应的是一个加载到JVM中的一个.class文件。通过Class可以得到一个类完整的结构。

实例化Class对象

  1. $\color{blue}{若已知具体的类，通过类的class属性获取，该方法最为安全可靠，程序性能最高。}$

    Class c = String.class;

  2. $\color{blue}{已知某个类的实例，调用该实例的getClass()方法获取Class对象}$

    Class c = p.getClass(); //p是一个对象

  3. $\color{blue}{已知一个类的全类名，且该类在类路径下，可通过Class类的静态方法forName()获取，可能抛出ClassNotFoundException}$

    Class c = Class.forName(“java.lang.String”);

获取类的全名，实现的接口等方法全部通过调用方法实现，本文不做具体举例。

使用反射的构造方法来创造对象

getDeclaredConstructor()获取类的全部构造方法
1.$\color{blue}{直接通过Class对象来创建}$
Objext o = c.newInstance(); //相当于调用类的无参构造
Person p = (Person)o;

在jdk9之后可以写成：

c.getDeclaredConstructor().newInstance();

反射机制可以强制的调用私有的构造方法：

Constructor cst = c.getDeclaredConstructor(参数列表.class);
cst.setAccessible(true); //解除私有的封装，对私有方法强制调用
Person p = (Peron)cst.newInstance(实参);

使用反射获取类的方法

public Method[] getDeclaredMethods();
返回此对象所表示的类或接口的所有方法；
public Method[] getMethods();
返回此对象所表示的类或接口的所有公共方法；

Method[] ms = c.getMethods();
Method[] ms = c.getDeclaredMethods(); //获取类的所有方法，包括共有和私有
for (Method m: ms){
    System.out.println("方法名" + m.getName());
    System.out.println("返回值类型" + m.getReturnType());
    System.out.println("修饰符" + m.getModifiers());
    Class[] pms = m.getParmeterTypes();//获取方法的参数类型
    if (pms != null && pms.length > 0){
        for (Class cs: pms){
            System.out.println("参数类型" + cs.getName());
        }
    }
}

使用反射获取类的属性

方法名称 作用
(Class) Field[] getFields() 返回此Class对象所表示的类或接口的public的Field
(Class) Field[] getDeclaredFields(); 返回此Class对象所表示的类或接口的全部Field
(Field) int getModifiers() 以整数形式返回此Field的修饰符
(Field) Class<?> getType() 得到Field的属性类型
(Field) String getName() 返回Field的名除 
Fileds[] fs = c.getFileds(); //获取类的共有属性，包括父类的属性
Fileds[] fs = c.getDeclaredFileds(); //获取所有（不包括父类）的属性（包括私有）
for (Field f: fs){
    System.out.println("属性的修饰符" + f.getModifiers());
    System.out.println("属性的类型" + f.getType());
    System.out.println("属性的名除" + f.getName());
}

使用反射获取类所在的包

Package p = c.getPackage();

通过反射调用类中的指定方法和属性

调用指定方法

通过Method类完成：

  1. $\color{blue}{通过Class类的getMethod方法取得一个Method对象，并设置参数}$

    Class c = Person.class;
Method m = c.getMethod("showInfo", String.class);
Method m1 = c.getDeclaredMethod("showName"); //访问私有方法
m1.setAccessible(true);//解除私有封装

  2. $\color{blue}{使用invoke进行调用}$

    Person p = (Person)c.getDeclaredConstucted(int.class, Sting.class).newInstance(10, "Q");
//可以直接设为Object不强制转换
m.invoke(p, "hello"); //参数一：需要实例化的对象，参数二(可变参数)：调用当前的方法的实参
m1.invoke(p);

  3. $\color{blue}{如果调用的函数有返回值}$
    假设showInfo方法返回一个int型的数据：

    int result = m.invoke(p, “hello”);

调用指定属性

可以通过Field类操作类中的属性，通过Field类提供的set()和get()方法就可以完成设置和取得属性内容的操作。

方法名称 作用
Field getField(String name) 返回此Class对象表示的类或接口指定的public的Field
void set(Object obj, Object value) 设定指定对象obj上此Field的属性内容 
Class c = Person.class;
Person p = (Person)c.getConstructor().newInstance();
Field f1 = c.getDeclaredField("val");
Field f2 = c.getDeclaredField("name");
f1.set(p, 10); //对对象p的val属性设置称10；
int result = (int)f1.get(p); //获取p对象的val属性的值
  • $\color{red}{注：}$
    • 如果访问私有属性，调用setAccessible解除私有封装。
    • 在示例类中val和name没有设置访问权限，那么这两个属性的访问权限为默认。如果出现java.lang.NoSuchFieldException错误，请手动把类中的属性声明为Public，否则请调用getDeclaredField方法。

动态代理

示例类

class Person {
    int val;
    String name;
    public Person(){ }

    public Person(int val){
        this.val = val;
    }

    public Person(String name){
        this.name = name;
    }

    public  Person(int val, String name){
        this.val = val;
        this.name = name;
    }
    public void showInfo(String n){
        System.out.println(this.name + this.val + n);
    }

    private void showName(){
        System.out.println("这是一个私有方法" + this.name);
    }
}
