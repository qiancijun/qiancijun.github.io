阶乘

四月 02, 2020
  • 算法

程序要求：用递归实现阶乘（不考虑数据溢出问题）

方法一：递归

Copy Code

int fac(int num){
    if(num == 0){
        return 1;
    }
    else{
        return num * fac(num - 1);
    }
}

在递归法中，由于使用的是尾递归，尾递归可以和循环进行转换，

方法二：循环

Copy Code

int fac(int num){
    if(num == 0){
        return 1;
    }
    int result = 1;
    for(int i = num; i > 0; i--){
        result *= i;
    }
    return result;
}

两个程序都只扫描了一遍，时间复杂度On
