前言

本文介绍的是排序算法中较常用的方法，归并排序，该方法的核心想法为分治法。分而治之，将原问题划分成多个子问题，子问题解决后原问题也随之解决。该算法不难实现，效率高，较稳定。学习归并算法之前最好了解递归。

思路

归并排序主要由两部分组成，第一部分是实现“并”，第二部分是实现“归”。实现“并”的过程中运用了双指针的方法，“归”的想法比较简单，不断的把数组对半分，直到数组里只有一个元素，一个元素的数组我们认为他默认有序。这是归并排序大致上的一个框架。

算法实现

合并数组

归并比较核心的部分就在这里，我们该这么把两段已经各自有序的数组重新归在一个数组内。这个方法也并不难想到。利用双指针遍历两个数组，每一次把两个指针指向的小的那个元素添加到数组里，然后指针向后移动，重复此操作，直到遍历完其中一个数组。最后把没遍历完的数组抄在新数组后。切记要检查原来被分割后的数组中是否还有剩余元素，不要遗漏。

分割数组

分割数组没有什么技巧，每次分割把数组划分为等长的两个，无限下去，直到数组内的元素只剩下一个。

void merge(vector<int> &nums, int L, int M, int R){
    int left_size = M - L + 1;
    int right_size = R - M;
    int i = 0, j = 0, k = L; //双指针
    vector<int> left(left_size, 0);
    vector<int> right(right_size, 0); //分割两数组
    for(int n = L; n <= M; n++)
        left[n - L] = nums[n];
    for(int n = M + 1; n <= R; n++)
        right[n - M - 1] = nums[n];
    //填充数组
    while (i < left_size && j < right_size){
        if (left[i] < right[j]){
            nums[k++] = left[i++];
        }
        else {
            nums[k++] = right[j++];
        }
    }//归并
    while (i < left_size){
        nums[k++] = left[i++];
    }
    while (j < right_size){
        nums[k++] = right[j++];
    }//检查是否有遗漏
}

void merge_sort(vector<int> &nums, int L, int R){
    if (R == L){
        return;
    } //退出条件，R == L 意味数组最左的下标等于数组最右边的下标，此时我们认为数组里只有一个元素
    else {
        int mid = (L + R) / 2;
        merge_sort(nums, L, mid); //对左边进行归并排序
        merge_sort(nums, mid + 1, R); //对右边进行归并排序
        merge(nums, L, mid, R); //把进行完归并排序的左右数组并一下
    }
}
