1403. 非递增顺序的最小子序列

思路

排序之后每次都往子序列里放一个最大的，直到子序列和大于原数组。

标签

贪心算法 排序

代码

Copy Code

vector<int> minSubsequence(vector<int>& nums) {
    vector<int> result;
    if (nums.size() < 2)
        return nums;
    sort(nums.rbegin(), nums.rend());//排序
    int sum1 = 0, sum2 = 0;
    for (int a: nums){
        sum1 += a;
    }
    for (int i = 0; i < nums.size(); i++){
        if (sum1 >= sum2){
            sum1 -= nums[i];
            sum2 += nums[i];
            result.push_back(nums[i]);
        }
        else {
            break;
        }
    }
    return result;
}

1404. 将二进制表示减到 1 的步骤数

思路

如果把二进制转换为十进制在操作，按理可以完成，不过给的二进制可能远远超过64位。我们需要直接在二进制字符串上操作。要解决的问题就是奇数情况下加一的进位问题，用一个bool记录一下是否要进位。

如果要进位，1进位变成0，0只需要一步那就是除以2。0进位变成1，1要加1再除以二就是两步，并且往前进位

如果不要进位，1加一除以2就是两步，往前进一位，0直接除以2就是一步，不需要进位。

最后处理到第一位判断一下之前是不是要进位。

标签

位运算 字符串

代码

Copy Code

int numSteps(string s) {
    int step = 0;
    int attach = 0;
    for (int i = s.size() - 1; i > 0; i--){
        if (attach == 1){
            if (s[i] == '0'){
                step += 2;
                attach = 1;
            }
            else {
                step++;
                attach = 1;
            }
        } //有进位
        else {
            if (s[i] == '0'){
                step++;
                attach = 0;
            }
            else {
                step += 2;
                attach = 1;
            }
        } //没有进位
    }
    if (attach == 1){
        step++;
    }
    return step;
}

1405. 最长快乐字符串

思路

直接贪心做，每次都取最多的那个，如果已经取了两次就取第二多的，在写的过程中注意个数就行了

标签

贪心算法 动态规划

代码

Copy Code

string longestDiverseString(int a, int b, int c) {
    vector<pair<int, char>> nums;
    nums.push_back(make_pair(a, 'a'));
    nums.push_back(make_pair(b, 'b'));
    nums.push_back(make_pair(c, 'c'));
    sort(nums.begin(), nums.end());
    string result = "";
    while (true){
        sort(nums.rbegin(), nums.rend());
        int len = result.size();
        if (len >= a + b + c)
            break;
        if (len < 2){
            result += nums[0].second;
            nums[0].first--;
        }
        if (len >= 2 && result[len - 1] == result[len - 2]){
            if (nums[0].second != result[len - 1] && nums[0].first > 0){
                result += nums[0].second;
                nums[0].first--;
            }
            else if (nums[0].second == result[len - 1] && nums[1].first > 0){
                result += nums[1].second;
                nums[1].first--;
            }
            else {
                break;
            }
        }
        else if (len >= 2 && result[len - 1] != result[len - 2] && nums[0].first > 0){
            result += nums[0].second;
            nums[0].first--;
        }
    }
    return result;
}
