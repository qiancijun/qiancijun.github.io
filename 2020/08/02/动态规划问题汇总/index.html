背包问题

01背包

有 $N $件物品和一个容量是 $V $的背包。每件物品只能使用一次。

第 $i $件物品的体积是 $vi$，价值是 $wi$。

求解将哪些物品装入背包，可使这些物品的总体积不超过背包容量，且总价值最大。
输出最大价值。

输入格式
第一行两个整数，$N$，$V$，用空格隔开，分别表示物品数量和背包容积。

接下来有 $N $行，每行两个整数 $vi$,$wi$，用空格隔开，分别表示第 $i $件物品的体积和价值。

输出格式
输出一个整数，表示最大价值。

数据范围
$ 0 < N, V ≤ 1000 $
$ 0 < Vi,Wi ≤ 1000 $

思路

$dp[i][j]$意为取前i个物品，且当前容量为j的最大价值
如果当前物品的体积小于当前容量，那么我们选择不取，当前答案为$dp[i][j] = dp[i - 1][j]$
否则我们可以选取，答案为$dp[i][j] = dp[i - 1][j - v[i]] + w[i]$

完全背包

有 $N$ 种物品和一个容量是 V 的背包，每种物品都有无限件可用。

第 $i$ 种物品的体积是 $vi$，价值是 $wi$。

求解将哪些物品装入背包，可使这些物品的总体积不超过背包容量，且总价值最大。
输出最大价值。

输入格式
第一行两个整数，$N$，$V$，用空格隔开，分别表示物品种数和背包容积。

接下来有 $N$ 行，每行两个整数 $vi$,$wi$，用空格隔开，分别表示第 $i$ 种物品的体积和价值。

输出格式
输出一个整数，表示最大价值。

数据范围
$0 < N, V ≤ 1000$
$0 < vi, wi ≤ 1000$

思路

和01背包很相似，无非就是取1个变成了取多个，其实可以在01背包的基础上，再加一层循环，判断最多可以取几个，但是时间复杂度会上升至$n^3$，会TLE。
于是我们需要进行优化
若按照刚才上述的想法，三层循环，我们可以得到这样一个状态转移方程
$dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - v[i]] + w[i], dp[i - 1][j - 2 v[i]] + 2 w[i], + dp[i - 1][j - 3 v[i]] + 3 w[i]…..)$
我们对他做一步递推
$dp[i][j - v]=max(dp[i - 1][j - v[i]], dp[i - 1][j - 2 v[i]] + w[i], dp[i - 1][j - 3 v[i]] + 2 * w[i] …..)$
两个方程存在一定关系，第一个方程从第二项和第二个方程从第一项只相差一个$w[i]$
于是得到一个新的状态转移方程
$dp[i][j] = max(dp[i][j], dp[i][j - v[i]] + w[i]);$
