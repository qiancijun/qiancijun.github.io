题目描述

给定一个字符串 s ，找到其中最长的回文子序列，并返回该序列的长度。可以假设 s 的最大长度为 1000 。
示例一
输入:

“bbbab”

输出:

4

一个可能的最长回文子序列为 “bbbb”。


示例二
输入:

“cbbd”

输出:

2

一个可能的最长回文子序列为 “bb”。

思路

一道典型的区间DP题
$dp[l][r]$意为，在区间$l$至$r$内最长回文子序列的个数
动态转移方程：如果$s[l] = s[r]$，则$dp[l][r] = dp[l + 1][r - 1] + 2$
否则$dp[l][r] = max(dp[l + 1][r], dp[l][r - 1])$
显然如果$l = r$，可认为单个字符即可构成长度为1的回文子序列
我们可以从长度为2的子序列进行遍历，知道了子序列长度，和左区间的值，我们可以推出右区间是多少$r = l + len - 1$
时间复杂度：$O(n^2)$
