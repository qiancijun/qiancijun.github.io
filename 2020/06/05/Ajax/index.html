Ajax

  • Ajax：ASynchronous JavaScript And XML。是指一种创建交互式、快速动态网页应用的网页开发技术，$\color{red}{无需重新加载整个网页的情况下，能够更新部分网页的技术}$。通过在后台与服务器进行少量数据交换，Ajax 可以使网页实现异步更新。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新。
    1. 异步和同步
      • 同步：客户端必须等待服务端的响应，在等待期间客户端不可以做其它操作
      • 异步：客户端不需要等待服务端的响应，在服务的处理请求的过程中，客户端可以进行其它的操作

实现方式

  1. 原生js方式（了解）

  2. jQuery方式

    • $.ajax()
      • 语法：$.ajax(url, [settings])
      • 语法：$.ajax({键值对})
    • $.get()：发送get请求
      • 语法：$.get(url, [data], [callback], [type]);
      • url：请求路径
      • data：请求参数
      • callback：回调函数
      • type：响应数据的类型
    • $.post()：发送post请求

  3. 其它参数

    • dataType：设置接受到的响应数据的格式

JSON

  • JavaScript Object Notation，是存储和交换文本信息的语法，类似XML， JSON比XML更小，更快，更易解析

语法

  1. 基本规则
    • 数据在名称/值对中：JSON数据是由键值对构成的
      • 键用引号包括，也可以不使用引号
      • 取值类型：数字（整数或浮点数、字符串（在双引号中）、逻辑值（true 或 false）、数组（在方括号中）、对象（在花括号中）、null
    • 数据由逗号分隔：多个键值对由逗号分割
    • 花括号保存对象：使用{}定义JSON格式
    • 方括号保存数组：使用[]定义数组
  2. 获取数据：
    1. json对象.键名
    2. json对象[“键名”]
    3. 数组对象[索引]
  3. JSON数据和Java对象的相互转换
    • JSON解析器：常见的解析器
      • Jsonlib
      • Gson：谷歌的解析器
      • fastjson：阿里巴巴的解析器
      • jackson：spring内置的解析器
    • JSON转为Java对象
      • readValue(json字符串数据, class)
    • Java对象转为JSON
      • 创建jackson核心对象 ObjectMapper
      • 调用ObjectMapper的相关方法
      • 注解：
        1. @JsonIgnore：排除属性
        2. @JsonFormat：属性值格式化
      • 复杂的Java对象：List、Map集合
