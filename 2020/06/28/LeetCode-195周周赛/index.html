5448. 判断路径是否相交

¶思路

  • 用一个只有两个元素的数组来模拟一个点，每次移动一步，对应坐标轴的值加一，把新的坐标点加入哈希表中，如果哈希表中已经存在了已有的点即为路过

5449. 检查数组对是否可以被 k 整除

¶思路

  • 利用余数！！！
  • 余数可以模拟一个循环，例如$\textit{k = 5}$，可以模拟一个0-5的循环，那么用一个长度为$\textit{k}$的数组来记录每个数余k后的值是多少
  • 负数处理：负数求余(num % k + k) % k
  • 答案判断：数组下标为0的值代表已经可以被k整除，只要保证该值是偶数，能两两配对就可以。其余判断两下标相等的值是否相等即可

5450. 满足条件的子序列数目

¶思路

  • 满足条件的答案为：最后一个大于target的值的下标减去遍历到元素的下标。在和2做幂运算。
  • 由于答案是几个2的幂的和，可以提前维护一个长度为nums数组长度的数组，来记录对应下标的2的幂。
