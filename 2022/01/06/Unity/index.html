Unity Core

Unity 版本 2020.3.25

¶创建项目

从 Hub 中创建一个 3D 的 Unity 项目；安装一系列必要的插件

搜索Universal Render Pipeline

  1. 安装完成后需要创建自己的 Pipeline Asset

    ![](E:\Study\MyNotes\Unity\URP Assets.png)

  2. edit -> project settings -> graphics 设置可编辑的渲染管线为urp

  3. edit -> project settings -> quality 修改默认的 renderer 为 urp

  4. edit -> render pipeline -> universal render pipeline -> upgrade project materials to UniversalRP Materirals 将项目升级为 URP 项目

¶修改skybox

使用的素材为Skybox Extended

window -> rendering -> Lighting -> Environment -> skybox material 修改默认的天空盒子

¶设置阴影参数

当在场景中添加地面时，人物素材就拥有了阴影；通过设置urp的Shadwos参数来调整阴影效果

¶添加素材及摄像机

添加素材时，按住V可以开启吸附功能，将素材吸附到地面。

选好视角，按ctrl + shift + f调整摄像机视角

¶Poly Brush

  1. 在 package manager 中安装 polybrush
  2. 在安装界面中点开samples根据项目导入shader

¶ProBuilder

​ 安装方法同poly brush，选择显示方式为Use Icon Mode

按住Alt键单机第一个Icon，可以在场景中新建Object，选择Plane，长宽为50，段数为30；这样建成的平面有一个缺点，其中心原点不在其几何中心，通过平移时可以看见。如果希望中心点在几何中心，可以点击probuilder中的center pivot来设定其中心

¶Progrid

该插件属于preview插件，在Unity2020版本之后，在package manager无法看见preview插件，需要手动开启

随后安装方式同ProBuilder

¶添加自己的材质

在Assets下新建Materials，将新建的材质直接拖入要对应的Object，就可以更改颜色

AI 导航

打开AI 烘焙导航；点击Object，设置其static可以改变其属性，walkable表示改模型是可以行走的；

点击玩家（player）在Inspector中add component -> navigation -> agent；随后选中player，按下f键锁定，根据模型修改宽和高（Navigation里也要修改）

¶Mouse Manager

  1. 在Assets里新建文件夹scripts，创建第一个脚本文件MouseManager.cs

  2. 编辑代码

  3. 在场景中新建一个空对象，将脚本文件拖入空对象，将要控制的物体拖入编写的事件中

¶使用Event实现点击走路

修改 MouseManager 使其变为单例模式

新建player controller，专门用于控制角色

将此controller用于player
